To Signup:

Use Developer Signup Page and STORE YOUR API KEY SAFELY!

To Create a Subscription

Use This Form or submit a POST request to http://mlbvsnba.no-ip.org:8000/createsub/ with:

  • api_key: string - your api key returned by signup
  • name: the name of the subscription list (limit is 50 chars and must be unique)
  • description: the description of the subscription list (limit is 300 chars)

To Create/Push a Notification to a Subscription

Use This Form or submit a POST request to http://mlbvsnba.no-ip.org:8000/pushnotif/ with:

  • api_key: string - your api key returned by signup
  • sub_id: int - the id of the subscription list that YOU OWN/CREATED and that you want to push the notification to
  • contents: string - the contents of your notification (limit is 300 chars)
  • url: string - optional - the url that should be associated with your notification
